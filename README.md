# 🏦 Bank Customer Churn Analysis

## 🎯 Цель проекта
Проанализировать поведение клиентов банка, выявить причины оттока, сегментировать аудиторию и построить модель машинного обучения, которая предсказывает уход клиента до того, как он произошёл.

## 🏢 Бизнес-задача
Банк теряет клиентов и не знает, кто уйдёт следующим. Единая стратегия удержания применяется ко всем — без учёта профиля риска. Необходимо выявить факторы оттока, разбить клиентов на сегменты и дать менеджерам инструмент для приоритизации работы.

## ❓ Бизнес-вопросы
1. Клиенты из каких стран уходят чаще и почему?
2. Влияет ли возраст и пол на вероятность оттока?
3. Как количество продуктов и активность клиента связаны с лояльностью?
4. Можно ли статистически подтвердить ключевые гипотезы об оттоке?
5. Какая модель лучше всего предсказывает уход клиента?

## 🗺️ План анализа

**Часть 1 — Загрузка и первичный осмотр**
- Размер, типы данных, уникальные значения
- Общий Churn Rate: 20.4% (2 037 из 10 000 клиентов)

**Часть 2 — Очистка данных**
- Удаление технических колонок и дубликатов
- Кодирование `Geography` и `Gender` через `LabelEncoder`
- Создание новых признаков: `Age Group`, `Balance Group`, `Churn Status`

**Часть 3 — Разведочный анализ (EDA)**
- Распределение оттока по странам, полу, возрасту
- Анализ количества продуктов: нелинейная зависимость с оттоком
- Влияние активности клиента на лояльность
- Корреляционная матрица числовых признаков

**Часть 4 — Сегментация (KMeans)**
- Метод локтя — выбор оптимального K = 4
- Визуализация кластеров через PCA (2D-проекция)
- Профиль и уровень оттока в каждом сегменте

**Часть 5 — Ключевые метрики**
- Сводная таблица Churn Rate по всем категориям
- Визуализация рейтинга факторов риска

**Часть 6 — Проверка гипотез (t-test)**
- H1: клиенты из Германии уходят значимо чаще, чем из Франции
- H2: неактивные клиенты уходят значимо чаще активных
- H3: клиенты старше 45 лет уходят значимо чаще молодых

**Часть 7 — Модели прогнозирования оттока**
- Logistic Regression — базовая линия
- Random Forest — ансамбль деревьев
- Gradient Boosting — финальная модель
- Метрики: AUC-ROC, Precision, Recall, матрица ошибок
- Важность признаков, 5-Fold Cross-Validation

**Часть 8 — Выводы и рекомендации**

---

## 📊 Ключевые выводы

### EDA
- **Германия** — Churn Rate **32.4%**, вдвое выше Франции и Испании
- **Клиенты 45–59 лет** — уходят в **49.4%** случаев (каждый второй)
- **Женщины** уходят на 8.6 п.п. чаще мужчин (25.1% vs 16.5%)

### Продукты и активность
- Клиенты с **3–4 продуктами** уходят в **83–100%** случаев — критический сигнал
- Клиенты с **2 продуктами** — самый лояльный сегмент (Churn Rate **7.6%**)
- **Неактивные** клиенты уходят вдвое чаще активных (26.9% vs 14.3%)

### Гипотезы (t-test, α = 0.05)
| Гипотеза | Результат | p-value |
|---|---|---|
| H1: Германия уходит чаще Франции | ✅ Подтверждена | < 0.0001 |
| H2: Неактивные уходят чаще активных | ✅ Подтверждена | < 0.0001 |
| H3: Клиенты 45+ уходят чаще молодых | ✅ Подтверждена | < 0.0001 |

### Модели
| Модель | AUC-ROC |
|---|---|
| Logistic Regression | 0.774 |
| Random Forest | 0.848 |
| **Gradient Boosting** 🏆 | **0.867** |

**Топ признаков по важности (Gradient Boosting):** Age (25.3%) → Balance (14.3%) → EstimatedSalary (14.0%) → CreditScore (13.4%) → NumOfProducts (13.6%)

**Рекомендации бизнесу:**
- Разработать отдельную retention-стратегию для Германии (Churn Rate 32%)
- Запустить программу удержания для клиентов 45+ (персональный менеджер, специальные условия)
- Пересмотреть продуктовые пакеты с 3+ продуктами — клиент не видит в них ценности
- Реактивировать неактивных клиентов через e-mail и push-кампании
- Внедрить модель Gradient Boosting в прод: ежемесячно скорить клиентов и передавать в CRM список «клиентов риска» с вероятностью оттока > 0.5
